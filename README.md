# URL shortener 

## FEATURES
* NodeJS
* Express
* mongoDB
* mongoose


## HOW TO START

```bash
1. 安装并启动mongoDB
2. git clone git@github.com:zenghj/urlShortener.git
3. cd urlShortener
4. npm install
5. npm run start
```
